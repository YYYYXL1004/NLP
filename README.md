# NLP 作业与练习提交

这是我的自然语言处理课程作业及平时练习的代码汇总。

## 目录结构说明

整个项目的文件架构如下：

```text
.
├── main.py                # 【作业1】主程序 (包含分词、词性标注、NER的完整评估与演示)
├── 实验报告.md             # 【作业1】实验报告
├── 平时练习报告.md         # 【平时练习】练习1和练习2的汇总报告
├── bad_cases.txt          # 【作业1】程序自动生成的错误案例分析
├── images/                # 【作业1】生成的评估图表 (混淆矩阵、F1对比图等)
├── data/                  # 数据集存放目录
│
├── ex1/                   # 【练习1】PyTorch 线性拟合练习
│   └── main.py            
│
└── ex2/                   # 【练习2】Word2Vec 词向量运算与分析
    └── main.py
```

## 如何运行

### 1. 运行作业1 (核心作业)
直接在根目录下执行：
```bash
python main.py
```
程序会自动完成所有任务的评估、绘图，并生成错误报告。最后会启动一个**交互演示系统**，您可以输入任意句子进行测试。

### 2. 运行平时练习
- **练习1** (模型训练):
  ```bash
  cd ex1
  python main.py
  ```

- **练习2** (Embedding 分析):
  ```bash
  cd ex2
  python main.py
  ```

## 环境依赖
- spaCy (需安装 `zh_core_web_sm` 模型)
- PyTorch
- Matplotlib
- Scikit-learn (用于绘制混淆矩阵)
